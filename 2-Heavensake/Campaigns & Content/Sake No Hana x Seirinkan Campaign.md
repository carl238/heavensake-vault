## Recommended Campaign Setup

Based on butterfly campaign performance analysis (March 30, 2026):

| Setting | Recommendation |
|---|---|
| Objective | Engagement |
| Conversion Location | On Your Ad |
| Engagement Type | Video Views |
| Performance Goal | **ThruPlay Views** (NOT Maximize engagement with a post) |
| Budget Type | Lifetime Budget ($300) |
| Schedule | 10-day flight |
| Age | 21–55 (cap at 55 — 55+ demographic significantly more expensive per view) |
| Advantage+ Audience | OFF (alcohol compliance) |
| Placements | Manual: IG Reels, IG Stories, FB Reels, FB Stories ONLY |

### Why ThruPlay (Not Engagement)

The butterfly engagement campaign proved that "Maximize engagement with a post" generates 99.92% low-quality interactions (3-second auto-plays counted as engagements). For a 30-second cinematic reel with narrative structure, ThruPlay ensures Meta finds viewers who will watch at least 15 seconds — absorbing the craft story, the HEAVENSAKE bottles, and the dining atmosphere.

See full analysis: [[Meta Ads Performance Analysis — March 2026]]

### Placement Rationale

Restrict to Reels + Stories only. The butterfly campaign ran across Feed, Explore, Threads, Notifications, Search Results — most of these display vertical 9:16 content poorly. The DaVinci Resolve production value only shines in native vertical environments.
