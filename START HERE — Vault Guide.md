## The Rules

1. **Everything enters through Inbox.** Don't file directly — dump it in `0-Inbox`, then sort during review.
2. **Link aggressively.** When a Carl thought connects to a Heavensake decision, link them with `[[double brackets]]`. This is the vault's superpower.
3. **Use templates.** Every note type has a template in `4-Resources/Templates`. Use them.
4. **Tag for cross-cutting themes.** Tags cut across folders:
   - `#carl-origin` — ideas that originated from Carl's thinking
   - `#founder-voice` — content written in Carl's authentic voice
   - `#brand-dna` — anything that defines what HEAVENSAKE IS
   - `#never-say` — language guardrails
   - `#campaign/sakura` `#campaign/10yr` — campaign-specific
   - `#market/us` `#market/japan` `#market/china` — market-specific
   - `#product/label-noir` `#product/prestige-one` etc.
5. **Properties first, tags second.** Use YAML frontmatter for structured data (type, status, product, date). Use tags for themes and connections.
